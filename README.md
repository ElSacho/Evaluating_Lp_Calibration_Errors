# A Variational Estimator for Lp Calibration Errors - Experiments

[![arXiv](https://img.shields.io/badge/arXiv-2602.24230-b31b1b.svg)](https://arxiv.org/abs/2602.24230)

This repository contains the experiment code and scripts to reproduce the results presented in our paper: **[A Variational Estimator for Lp Calibration Errors](https://arxiv.org/abs/2602.24230)**. 

## 📦 Corresponding Package: [`probmetrics`](https://github.com/dholzmueller/probmetrics)

> **🚨 IMPORTANT:** If you are looking to use the $L_p$ calibration error estimators in your own projects, we highly recommend using our dedicated Python package rather than the raw experiment code in this repository. 

👉 **Get the package here:** [**github.com/dholzmueller/probmetrics**](https://github.com/dholzmueller/probmetrics)

---

## 🔬 Reproducing the Experiments

This repository is kept primarily for transparency and reproducibility of the figures and tables in the paper.

You can reproduce our experiments by running the `code/generate_experiment_{experiment_name}.py` files.

Please cite us if you use our findings.
